本章我们主要用到project模块和debug模块。

project模块包含用于管理和执行项目操作的函数。例如，通过函数get_value和get_values我们可以取得一个对象的某个/某些特定参数的值。同样的，使用set_value和set_values我们可以设定一个对象的某个/某些特定参数的值。

debug模块包含用于管理和执行日志输出的函数，例如log和warning。
